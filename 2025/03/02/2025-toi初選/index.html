2025-03-02

前言

其實在幾個月前，我就有嘗試架我的blog(只是都沒有很完整)，現在經過ioicamp+toi初選的洗禮，決定架這個blog，雖然不會因為架了blog就變電神，但我就是想架，未來會不定期po關於競程的紀錄

題目 略

以下是大暴雷

開場

先來po記分板 (個人 rk 131/364)(看來1J沒望了)

00:05 A 0

讀完A，本想說就直接做(結果因為這個決定，導致我第一個AC submission隔了約2小時，原因我不會Linux也不會再用vs code的terminal外加debug)

00:53 C 0

心態崩了，簽到題沒拿到，於是就看了B像盒內塔，就直接跳過看接下來的C題，想一下發現第2子題可用stack維護，於是一發下去WA

01:43 E 7

本想說我圖論超強(我會dijkstra, bellman, floyd…)，結過這題直接嗆我，我不會建圖，只寫出第1子題，我相信正常人都一定想的到

01:45 A 0

嘗試A許久才發現我讀錯題目了:(，將取連續數字1在最大長度計算成連續數字1最長前綴

01:58 A 100

改一下，終於拿到第一個AC(其實，在此刻就知道我已經沒料了)

02:08 B 38

再改一下，耶，拿到第2子題38分(賽後才發現這題超多人AC)

02:58 D 5

在B、C打轉許久，才硬著頭皮閱讀完D，觀察一下，就發現第1子題就只是國小的數學題罷了

感想
